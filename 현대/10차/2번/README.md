## ⚠️ 해결 시 주의해야 할 핵심 포인트

### 1) 핵심 아이디어: 역발상 조각 조합 (Piece Composition)
이 문제는 8칸의 연결된 덩어리를 먼저 찾으려 하면 연산량이 폭증하여 시간 초과가 발생합니다 (${}_{16}C_8 = 12,870$개 조합마다 내부 탐색 발생). 대신 **"5칸짜리 조각(펜토미노) 2개를 찾고, 그 둘이 2칸 겹치게 배치하면 무조건 연결된 8칸이 된다"**는 성질을 이용해 탐색 범위를 획기적으로 줄여야 합니다.

* **핵심:** 8칸을 뽑고 다시 쪼개는 방식보다, 5칸 조각을 먼저 모두 추출한 뒤 2개씩 골라 비교하는 방식이 훨씬 효율적입니다.
* **➡️ 해결 방법:** 모든 가능한 5칸 연결 조각을 미리 추출(Step 1) -> 2개씩 골라 겹치는 칸 확인(Step 2).

### 2) Step 1: BFS를 활용한 5칸 펜토미노 추출
격자판($N \times M$)의 모든 칸 중 5개를 뽑는 조합(${}_{NM}C_5$)을 생성하고, 이들이 상하좌우로 연결되어 있는지 확인합니다.

* **핵심:** **BFS(Breadth-First Search)**를 사용하여, 뽑힌 5개의 좌표 중 한 점에서 시작해 모든 칸에 도달 가능한지 체크하여 '연결된 조각'만 리스트에 담습니다.
* **➡️ 해결 방법:**
    ```python
    # 5칸 조합 중 연결된 것만 valid_pentos에 저장
    for combo in combinations(cells, 5):
        nodes = set(combo)
        q = deque([combo[0]])
        visited = {combo[0]}
        while q:
            r, c = q.popleft()
            for i in range(4):
                nr, nc = r + dr[i], c + dc[i]
                if (nr, nc) in nodes and (nr, nc) not in visited:
                    visited.add((nr, nc))
                    q.append((nr, nc))
        if len(visited) == 5:
            valid_pentos.append(nodes)
    ```

### 3) Step 2: `set` 연산자(`&`)를 통한 광속 겹침 검증
추출된 수백 개의 펜토미노 조각들 중 2개를 선택했을 때, **"정확히 2칸 겹침"** 조건을 가장 빠르게 확인해야 합니다.

* **핵심:** 파이썬의 `set` 자료형에서 **`&` 연산자**는 교집합을 구하는 연산자로 오버로딩되어 있습니다. 이는 리스트 반복문보다 압도적으로 빠른 성능을 보여줍니다.
* **➡️ 해결 방법:**
    ```python
    # 두 조각 p1, p2의 교집합(겹치는 칸) 확인
    common = p1 & p2 
    if len(common) == 2:
        # 점수 계산: p1 합 + p2 합
        # 겹친 2칸은 p1, p2 양쪽에 모두 포함되어 있어 자동으로 2배 합산됨
        current_total = score_p1 + score_p2
        best_max = max(best_max, current_total)
    ```

### 4) Step 3: 효율적인 점수 합산 (Overlap Double Counting)
문제에서 "겹치는 2칸은 점수를 두 번 더한다"라고 명시되어 있습니다.

* **핵심:** 각 5칸 조각의 점수 합을 미리 계산해둔 뒤, 2칸 겹침이 확인되었을 때 단순히 두 점수를 더하면 겹친 좌표의 점수는 자연스럽게 결과값에 두 번 포함됩니다. 별도의 조건문 없이 산술 합만으로 최적의 효율을 도출할 수 있습니다.
