# [삼성 기출] 마법의 숲 탐색

> **문제 링크**: [코드트리 - 마법의 숲 탐색](https://www.codetree.ai/ko/frequent-problems/samsung-sw/problems/magical-forest-exploration/description)

---

## ⚠️ 구현 시 주의해야 할 핵심 포인트

### 1) 골렘의 이동 우선순위와 "회전 경로" 체크
골렘은 단순히 아래로 내려가는 것이 아니라 **[남쪽 ➡️ 서쪽 회전 ➡️ 동쪽 회전]** 순서로 움직입니다. 특히 서쪽/동쪽 회전 시에는 1칸 옆으로 밀린 뒤 아래로 한 칸 내려가기 때문에 이동 경로상에 위치한 칸들을 꼼꼼하게 확인해야 합니다.

* **오류 예시:** 단순히 이동 후의 중심 좌표 기준 십자($+$) 모양만 체크할 경우, 대각선 방향에 있는 장애물과 겹치는 현상이 발생합니다.
* **핵심:** 골렘의 현재 위치 $(r, c)$를 기준으로 회전 경로를 포함한 5개의 좌표가 비어있는지 확인해야 합니다.
* **➡️ 해결 방법:**
    * `can_west(r, c)`: `(r-1, c-1), (r, c-2), (r+1, c-1), (r+1, c-2), (r+2, c-1)` 좌표 확인.
    * `can_east(r, c)`: `(r-1, c+1), (r, c+2), (r+1, c+1), (r+1, c+2), (r+2, c+1)` 좌표 확인.



### 2) 숲의 여백(Top Margin)과 격자 초기화 조건
골렘은 숲 외부에서 진입하므로 실제 숲 상단에 여백 행이 필요하며, 안착 후 상태에 따라 숲을 비워야 합니다.

* **오류 예시:** 안착한 골렘의 중심 좌표만 보고 숲 내부 여부를 판단하면, 골렘의 윗부분이 숲 밖에 걸쳐 있는 경우를 놓치게 됩니다.
* **핵심:** 상단에 3행의 여백을 두어 `(R+3) x C` 격자를 운영하고, 최종 위치의 중심 행 `curr_r`이 3 이하라면 골렘의 일부가 숲 밖에 있는 것이므로 초기화합니다.
* **➡️ 해결 방법:**
    * `if curr_r <= 3:` 조건 만족 시 `arr[:] = [[0]*C for _ in range(R+3)]`를 수행하여 숲을 비우고 해당 정령의 점수는 누적하지 않습니다.

### 3) 정령의 이동(BFS) 시 "출구 환승" 로직
골렘 안착 후 정령은 연결된 모든 골렘 중 가장 남쪽 행으로 이동해야 합니다.

* **핵심 규칙:**
    1.  같은 골렘 내부는 자유 이동.
    2.  현재 밟은 칸이 **출구**일 때만 인접한 다른 골렘으로 이동 가능.
* **➡️ 해결 방법:**
    * 격자에 골렘을 마킹할 때 몸통은 `number`, 출구는 **음수(`-number`)**로 표시하여 구분합니다.
    * BFS 탐색 시 `abs(arr[curr_r][curr_c]) == abs(arr[nr][nc])`이면 같은 골렘 내 이동으로 처리합니다.
    * `arr[curr_r][curr_c] < 0` (출구)이고 `arr[nr][nc] != 0`이면 다른 골렘으로 넘어갈 수 있습니다.



### 4) 파이썬 리스트 가변(Mutable) 특성과 참조의 이해
함수 인자로 전달된 리스트를 초기화할 때 단순 할당과 슬라이싱 할당의 차이를 정확히 이해해야 합니다.

* **핵심:** `arr = []`는 함수 내부의 지역 변수 이름표만 바꾸는 것이지만, `arr[:] = []`는 원본 리스트의 메모리 주소를 유지한 채 내용물만 교체합니다.
* **➡️ 해결 방법:**
    * 숲을 초기화할 때 `arr[:] = [[0]*C for _ in range(R+3)]` 형식을 사용하거나 요소를 직접 `0`으로 수정하여 메인 루프의 배열에 즉시 반영되도록 합니다.

---

### 💡 정리하며
이 문제는 **복합 시뮬레이션(골렘의 우선순위 이동)**과 **그래프 탐색(정령의 환승 이동)** 능력을 동시에 요구합니다. 특히 골렘이 회전할 때 체크해야 하는 5개의 좌표를 정확히 산출하는 것이 안정적인 통과의 핵심이었습니다. BFS 탐색 시 출구를 음수로 마킹하는 테크닉은 다른 골렘으로의 전이 조건을 단순화하여 구현 실수를 줄여주었습니다.
